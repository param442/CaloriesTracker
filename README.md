# Calorie Tracker App

I am using turbo monorepo to manage my projects. This repository contains a calorie tracker app built with React and TypeScript. The app allows users to log their meals, track their calorie intake, and view insights about their nutrition.

# Technologies Used

- React
- TypeScript
- Tailwind CSS
- Turbo Monorepo
- Nest js (for backend API)
- Prisma (for database management)
- PostgreSQL (as the database)

# Features

- User authentication (sign up, log in, log out)
- Meal logging (add meals with calorie information)
- Daily and weekly calorie tracking
- Insights and analytics about nutrition
- Responsive design for mobile and desktop

# Getting Started

To run the app locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone
   ```
2. Navigate to the project directory:
   ```bash
   cd calorie-tracker
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and go to `http://localhost:3000` to see the app in action.
